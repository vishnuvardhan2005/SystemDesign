## Variables
There are 3 ways to define variables in JS  
var  
let  
const  
var is function scoped  
let and const are block scoped  
const cannot be re-assigned  

## Data types
Primities  
  Number  
  String  
  Bool  
  Undefined - no value assigned  
  Null  
  Symbol  
  BigInt  

Object types  
  Object  
  Array  

In JS anything aprt from primitives is an Object
